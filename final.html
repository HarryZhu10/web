Project Description:

The purpose of our project was to create a Python function that made it easier to sort CSV files and turn them into HTML. The function will have two inputs. The first will be the CSV document and the second one will be the tags of each file. The output will be a string that contains the HTML code. The contents of the HTML code will be the tags, links, and previews. You will be able to run the HTML code and it will create a website. The website will consists of all the tags, links of each CSV file, and the preview of it. The code makes it so much easier for someone to look for a specific file within a folder. The previews to the links will allow the person to see if they are going to the correct website.

How to Use:

Run the program in the same folder containing template.html, directorystyling.css, and the csv document. It will prompt two things in the shell, the name of the folder containing the files and the search query. Once they have been entered, the program will generate an html document named dir.html, which displays the results of the search.

How it Works:

The python code consists of a bunch of different functions that works together and return a string that contains the HTML code for the CSV files provided. The python code will require you to import the csv files that you want to sort. The "makedict" function will take the CSV files and break them into lists by removing the "\n." It will also place the given tags into a dictionary. The "search" function compares each dictionary value to query and adds it to list. The "makeheader" function creates the start of the HTML code. It also connects the code to a CSS styling page. The "makelinks" function returns the string that contains the link of each CSV file. It adds on to the ouput of the "makeheader" function.
